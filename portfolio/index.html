How I work

I’m early in my design journey. Right now I’m taking SEG3125 — Analysis and Design of User Interfaces, where I’m learning the principles behind interfaces that communicate clearly. My approach is simple: learn a principle, apply it deliberately, then look back and ask whether it actually made the experience easier.

→

Learn

Study the principle — scale, hierarchy, balance, contrast, colour, type — from trusted sources.

→

Apply

Make deliberate choices and write down why, so design isn’t accidental.

→

Build

Turn the design into a working interface with HTML, CSS, JavaScript and Bootstrap.

→

Reflect

Test it against the principles and feedback, then refine. Nothing here is final.

What I’m learning from

  • Nielsen Norman Group UX research & visual-design principles
  • Bootstrap 5 & W3Schools HTML, CSS, JavaScript & UI components
  • SEG3125 @ uOttawa Analysis & Design of User Interfaces
